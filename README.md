# UndefinedTaste - Restaurant Website

A modern, responsive restaurant website built with React, TypeScript, and Tailwind CSS. This project showcases a refined dining experience with an elegant user interface and smooth navigation.

## Project Overview

**UndefinedTaste** is a contemporary restaurant website featuring:
- Modern, sophisticated design with a dark theme
- Smooth page transitions and animations
- Fully responsive layout
- Easy navigation through different sections
- Information about the restaurant, menu, and services

## Tech Stack

- **Frontend Framework**: React 19.2.0
- **Language**: TypeScript 5.9.3
- **Build Tool**: Vite 7.2.4
- **Styling**: Tailwind CSS 3.4.17
- **Routing**: React Router DOM 7.11.0
- **CSS Processing**: PostCSS 8.5.6
- **Node.js**: Latest stable version


## Features

- **Home Page**: Landing page with restaurant branding and call-to-action buttons
- **About Page**: Information about UndefinedTaste's philosophy and story
- **Menu Page**: Restaurant menu display
- **Responsive Navbar**: Navigation with hover effects and smooth transitions
- **Dark Theme**: Modern dark UI with elegant color scheme (#191528, #00253e, #f4f4f9)
- **Smooth Animations**: Page transitions and hover effects

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Restaurant/Frontend
```

2. Install dependencies:
```bash
npm install
```

## Available Scripts

- **Development Server**:
```bash
npm run dev
```
Starts the Vite development server with hot module replacement (HMR).

- **Build for Production**:
```bash
npm run build
```
Compiles TypeScript and builds the project for production.

- **Preview Build**:
```bash
npm run preview
```
Preview the production build locally.

- **Lint Code**:
```bash
npm run lint
```
Run ESLint to check code quality.

## Color Scheme

- **Primary Dark**: `#191528` - Main background

## Getting Started

1. Install dependencies and start the development server:
```bash
npm install
npm run dev
```

2. Open [http://localhost:5173](http://localhost:5173) in your browser

3. Navigate through the pages using the navbar

## Development Guidelines

- **Components**: Place reusable components in `src/components/`
- **Pages**: Add new pages in `src/components/pages/`
- **Styling**: Use Tailwind CSS classes for styling
- **Types**: Leverage TypeScript for type safety
- **Routing**: Update routes in `App.tsx`
